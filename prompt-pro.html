About the App

Transform your iOS device into a professional teleprompter. Perfect for creators, speakers & presenters. Import scripts, customize speed & appearance, and share instantly.

Professional Teleprompter

  • Smooth, adjustable scrolling speed (30-300 words per minute)
  • High-contrast display for easy reading
  • Support for both portrait and landscape orientation
  • Customisable text size and font selection
  • Adjustable text and background colours

Script Management

  • Create and edit scripts directly in the app
  • Import text from Notes app or files
  • Organise multiple scripts in one place
  • Quick edit and delete functionality
  • Share extension for easy importing from other apps

Smart Controls

  • Intuitive play/pause button
  • Draggable control panel for comfortable positioning
  • Quick access to speed adjustments
  • One-tap font size controls
  • Instant colour scheme changes

Optimised for Performance

  • Maintains smooth scrolling even with long scripts
  • Saves all settings automatically
  • Remembers scroll position between sessions
  • Fast script switching
  • Efficient memory management

Perfect for:

  • YouTube creators
  • Public speakers
  • Presenters
  • Educators
  • Podcasters
  • Video producers
  • Anyone needing to read scripts professionally
